# 服务条款

**最后更新日期：** 2025-03-24

**生效日期：** 2025-03-24

## 一、用户义务

1. **禁止生成内容类型**：

   - 淫秽色情素材
   - 种族歧视内容
   - 暴力恐怖元素
   - 侵犯他人肖像权/知识产权的内容

2. **使用限制**：
   - 不得通过自动化脚本进行高频次请求
   - 免费用户每日生成次数不超过[XX]次

1. 用户输入的提示词(prompt)版权归属用户本人
2. 本站AI生成的图片默认授予用户非独占使用权
3. 平台保留展示匿名化生成案例的权利

## 三、免责条款

- 不保证生成结果100%符合预期
- 因网络故障导致的生成失败不承担赔偿责任
- 用户自行承担生成内容传播的法律责任

## 四、服务变更

保留调整以下事项的权利：

- 免费用户的额度限制
- 订阅套餐价格
- 技术支持响应时间

如检测到滥用行为，有权立即暂停/终止账户且不退还费用
